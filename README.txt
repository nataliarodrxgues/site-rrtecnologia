# Vitrine Online: R. R. Tecnologia



O objetivo desse projeto é facilitar o acesso dos clientes a produtos sem a necessidade de deslocamento físico até a região central, melhorando a segurança e comodidade. Com a utilização de ciência de dados para identificar e minimizar os riscos e inconvenientes enfrentados pelos moradores ao se deslocarem até a região central, promovendo uma solução que mantém a atividade comercial de maneira eficiente e segura.
